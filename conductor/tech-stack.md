# Technology Stack

## Core Technologies
- **Python**: Primary programming language.
- **Flet**: Framework for building the interactive desktop UI.
- **PyMuPDF (fitz)**: Engine for high-performance PDF manipulation and watermarking.
- **Pillow (PIL)**: Library for image processing and metadata stripping.
- **PyInstaller**: Tool for packaging the application into a standalone macOS app.

## Target Platforms
- **macOS**: Primary target operating system.
